Introdução ao html

HTML (HyperText Markup Language) é a linguagem fundamental para a criação de páginas web, usada para estruturar o conteúdo e apresentar informações de maneira organizada e acessível. Não é uma linguagem de programação, mas sim uma linguagem de marcação, que define a estrutura e o significado do conteúdo na web, utilizando elementos, tags e atributos

Formulário em html

Formulários HTML são usados para coletar dados dos usuários em websites. Eles são uma parte essencial para a interação com o usuário e o envio de informações para o servidor. Elementos importantes: form: Define o formulário e contém todos os seus elementos. input: Define campos de entrada, como campos de texto, senhas, botões de rádio, caixas de seleção, etc. textarea: Define um campo de texto multilinha para entradas maiores. label: Define um rótulo para um elemento de formulário. button: Define um botão clicável. fieldset e legend: Agrupa campos relacionados e fornece um título para o grupo.

Html semântico

O HTML semântico é uma forma de escrever código HTML que prioriza o significado e a estrutura do conteúdo, em vez de apenas a sua apresentação visual. Em vez de usar elementos genéricos como div e span para definir seções, o HTML semântico utiliza tags como header, nav, main, article, aside e footer para indicar a função de cada parte da página. Isso torna o código mais organizado, legível e acessível, tanto para humanos quanto para máquinas como navegadores e mecanismos de busca.

Tabela em html

Em HTML, tabelas são usadas para organizar dados em linhas e colunas, facilitando a apresentação de informações de forma clara e estruturada. As tabelas são definidas usando a tag table, e dentro dela são usadas as tags tr para criar linhas e td para criar células de dados. As tags th também podem ser usadas para definir células de cabeçalho.

Formatações em html

Em HTML, a formatação de texto serve para controlar a aparência visual do conteúdo da página, como negrito, itálico, tamanhos e estilos de fonte. Isso é feito através de tags HTML, que instruem o navegador sobre como exibir o texto. 
Formatação de texto em HTML:
Negrito: "<"b>..."<"/b> ou "<"strong>"<"/strong>
Itálico: "<"i>...""<"/i> ou "<"em>..."<"/em>. 
Sublinhado: "<"u>..."<"/u>. 
Títulos: "<"h1>..."<"/h1> até ""<"h6>..."<"/h6>, onde "<"h1> é o maior e "<"h6> o menor. 
Parágrafos: "<"p>..."<"/p>. 
Quebra de linha: "<"br>. 
Texto pré-formatado: "<"pre>..."<"/pre> exibe o texto exatamente como digitado, incluindo espaços e quebras de linha. 
Outros: "<"sub>..."<"/sub> (subscrito), "<"sup>..."<"/sup> (sobrescrito), "<"ins>..."<"/ins> (inserido), "<"del>..."<"/del> (excluído), entre outros.
